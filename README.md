# Caja registradora

---

## 1. Descripción del proyecto
Es una aplicación de escritorio intuitiva ideada como una caja registradora digital pensada para bares o cafeterías.

Permite gestionar:
- Productos por categoría
- Tickets
- Cobros
- Historial de ventas
- Edición de precios
- Base de datos
